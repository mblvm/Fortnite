# 👑 Королевская Битва 3D

Браузерная **3D** battle royale в стиле Fortnite (WebGL / Three.js).
Всё локально, без CDN и сборки — просто откройте `index.html` в браузере.

## Как играть

Вы против 15 ботов. Прыгайте с автобуса, лутайтесь, стройте и выживайте —
последний живой получает **Королевскую победу**!

| Клавиша | Действие |
|---|---|
| **Мышь** | камера и прицел (клик по экрану — захват мыши) |
| **WASD** | движение относительно камеры |
| **ЛКМ** | стрелять / бить киркой |
| **ПКМ** | построить стену перед собой (10 материалов) |
| **E** | подобрать лут / открыть сундук |
| **R** | перезарядка |
| **1–5** | слоты оружия |
| **Пробел** | прыгнуть с автобуса |
| **M** | вкл/выкл звук |
| **Esc** | освободить мышь |

## Что внутри

- 🎮 **Полное 3D** — вид от третьего лица, тени, туман, остров в океане
- 🚌 **Автобус и высадка** — выбирайте точку прыжка и планируйте на парашюте
- 🌩️ **Шторм** — фиолетовая стена зоны сужается в 5 этапов, снаружи — урон
- 🔨 **Строительство** — добывайте материалы киркой из деревьев и камней, ставьте стены
- 🔫 **Оружие с редкостью** — пистолет, ПП, дробовик, автомат, снайперка; от обычного до легендарного
- 📦 **Лут** — сундуки с открывающейся крышкой, аптечки, зелья щита
- 🤖 **Боты** — лутаются, воюют друг с другом, строят стены под огнём и убегают от шторма
- 🗺️ Миникарта, килфид, 3D-цифры урона, имена над головами, звуковые эффекты

## Запуск

```
открыть index.html
```

Никаких сборок и серверов не нужно — весь код в одном HTML-файле,
Three.js лежит рядом (`three.min.js`), интернет не требуется.
