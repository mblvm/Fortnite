# 👑 Королевская Битва 3D

Браузерная **3D** battle royale в стиле Fortnite (WebGL / Three.js).
Всё локально, без CDN и сборки — просто откройте `index.html` в браузере.

## Как играть

Вы против 15 ботов. Прыгайте с автобуса, лутайтесь, стройте и выживайте —
последний живой получает **Королевскую победу**!

| Клавиша | Действие |
|---|---|
| **Мышь** | камера и прицел (клик по экрану — захват мыши) |
| **WASD** | движение относительно камеры |
| **ЛКМ** | стрелять / бить киркой |
| **ПКМ** | построить стену перед собой (10 материалов) |
| **E** | подобрать лут / открыть сундук |
| **R** | перезарядка |
| **1–5** | слоты оружия |
| **6 / 7** | музыкальные эмоции: танец «Фонк-байкер» / «Гридди» |
| **Пробел** | прыгнуть с автобуса |
| **M** | вкл/выкл звук |
| **Esc** | освободить мышь |

## Что внутри

- 🎮 **Полное 3D** — вид от третьего лица, тени, туман, остров в океане
- 🙂 **Прорисованные персонажи** — туловище, руки, ноги, волосы, рюкзак и живое лицо
  (глаза со зрачками, брови, рот); анимация ходьбы, моргание, замах киркой
- 🚌 **Автобус и высадка** — выбирайте точку прыжка и планируйте на парашюте
- 🌩️ **Шторм** — фиолетовая стена зоны сужается в 5 этапов, снаружи — урон
- 🔨 **Строительство** — добывайте материалы киркой из деревьев и камней, ставьте стены
- 🔫 **Оружие с редкостью** — пистолет, ПП, дробовик, автомат, снайперка; от обычного до легендарного
- 📦 **Лут** — сундуки с открывающейся крышкой, аптечки, зелья щита
- 🤖 **Боты** — лутаются, воюют друг с другом, строят стены под огнём и убегают от шторма
- 💃 **Музыкальные эмоции** — два танца с музыкой, синтезируемой на WebAudio; боты празднуют убийства танцем
- 🗺️ Миникарта, килфид, 3D-цифры урона, имена над головами, звуковые эффекты

## Эмоции персонажей

У каждого персонажа (игрока и ботов) есть готовый риг лица: брови, глаза со
зрачками и рот плавно меняют форму. Сменить выражение можно одной строкой —
где угодно в коде игры:

```js
state.player.emotion = 'happy';   // или setEmotion(бот, 'angry')
```

Доступные эмоции: `neutral`, `happy`, `sad`, `angry`, `surprised`.

Новая эмоция добавляется одной строкой в таблицу `EMOTIONS` в `index.html`:

| Параметр | Что делает |
|---|---|
| `browY` | подъём бровей |
| `browTilt` | наклон бровей (`+` злость, `−` грусть) |
| `eye` | раскрытие глаз (прищур/широко) |
| `pupil` | размер зрачков |
| `mouth` | форма рта: `smile` / `frown` / `flat` / `o` |
| `mSize` | размер рта |

## Музыкальные эмоции (танцы)

Нажмите **6** или **7** — персонаж уберёт оружие, повернётся к камере и
станцует под музыку, которая синтезируется прямо в браузере (WebAudio,
никаких аудиофайлов):

- **6 — «Фонк-байкер»** 🏍 — серия ударов кулаками и руки к шлему под
  тяжёлый фонк: 84 BPM, ре-диез минор, 808-бас со слайдом и ковбелл-рифф
- **7 — «Гридди»** ⚽ — бег на месте с высокими коленями и махами рук
  под задорный рифф на 126 BPM

Повторное нажатие той же клавиши — выключить. Движение, выстрел, стройка
или полученный урон тоже прерывают танец. Боты после убийства иногда
празднуют случайным танцем — подойдите ближе, и музыку будет слышно.

Из кода танец запускается одной строкой: `startEmote(state, entity, 'griddy')`.
Новый танец добавляется в таблицу `EMOTES` (поза — в `dancePose`,
музыка — в `MUSIC_TRACKS`).

## Запуск

```
открыть index.html
```

Никаких сборок и серверов не нужно — весь код в одном HTML-файле,
Three.js лежит рядом (`three.min.js`), интернет не требуется.
